# Leaderboards Background Assets

## Background Image
Save your leaderboards background image here:
- **File path:** `public/leaderboards/background.webp` (or `.jpg`, `.png`)
- **Recommended format:** WebP for best compression, or JPG for compatibility
- **Recommended resolution:** 1920x1080 or higher
- **File size:** Optimize to under 1MB for fast loading

## Usage
The background will be applied to the Leaderboards page with appropriate dark overlays for content readability.

## Current Setup
The Leaderboards page now uses `/leaderboards/background.webp` as its background image.
