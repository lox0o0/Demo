# Landing Page Background

Place your landing page background image here.

**Recommended formats:**
- `.webp` (preferred for web)
- `.jpg` or `.jpeg`
- `.png`

**Usage in code:**
The background image is referenced in `components/LandingPage.tsx` as:
```tsx
src="/landing/landing-background.webp"
```

**Recommended dimensions:**
- 1920x1080 or larger
- Landscape orientation
- Optimized for web (compressed)
