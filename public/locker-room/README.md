# Locker Room Background Assets

## Background Image
Save your locker room background image here:
- **File path:** `public/locker-room/background.jpg` (or `.png`, `.webp`)
- **Recommended format:** JPG or WebP for best compression
- **Recommended resolution:** 1920x1080 or higher
- **File size:** Optimize to under 1MB for fast loading

## Usage
The background will be applied to the Locker Room (Dashboard) page with appropriate overlays for content readability.
