# Auth Provider Logos

## Google Logo
Save the Google logo here:
- **File path:** `public/auth-logos/google.svg` (or `.png`)
- **Recommended format:** SVG for scalability
- **Size:** 32x32px or larger

## Apple Logo
Save the Apple logo here:
- **File path:** `public/auth-logos/apple.svg` (or `.png`)
- **Recommended format:** SVG for scalability
- **Size:** 32x32px or larger
- **Note:** Apple logo should be white/light colored for dark backgrounds

## Usage
These logos are used in the sign-in screen for Google and Apple authentication buttons.
