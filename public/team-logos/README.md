- `panthers.svg` (or `.png`)

- `warriors.svg` (or `.png`)
- `cowboys.svg` (or `.png`)
- `titans.svg` (or `.png`)
- `dragons.svg` (or `.png`)
- `knights.svg` (or `.png`)
- `raiders.svg` (or `.png`)
- `tigers.svg` (or `.png`)
- `bulldogs.svg` (or `.png`)
- `dolphins.svg` (or `.png`)
