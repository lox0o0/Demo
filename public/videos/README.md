# Background Videos for Home Screen

## File Location
Save your background video files directly in this directory: `/public/videos/`

## Recommended File Naming
Use descriptive names like:
- `home-bg-1.mp4`
- `home-bg-2.mp4`
- `home-bg-3.mp4`
- `home-bg-4.mp4`
- `home-bg-5.mp4`
- `home-bg-6.mp4`

Or more descriptive names:
- `stadium-action-1.mp4`
- `player-highlights-1.mp4`
- `crowd-celebration-1.mp4`
- etc.

## Video Format Recommendations
- **Format:** MP4 (H.264 codec recommended for best browser compatibility)
- **Resolution:** 1920x1080 (Full HD) or 1280x720 (HD) - avoid 4K to keep file sizes reasonable
- **Aspect Ratio:** 16:9 (standard widescreen)
- **File Size:** Try to keep each video under 10-15MB for fast loading
- **Duration:** 10-30 seconds per video (they'll loop, so shorter is better)
- **Framerate:** 24-30fps is sufficient for background videos

## Implementation
The videos will be randomly selected and looped as background elements on the home screen. They'll play muted and automatically loop.
